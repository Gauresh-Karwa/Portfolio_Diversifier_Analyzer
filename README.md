# Portfolio Diversification Analyzer :

A full-stack web application that enables users to analyze portfolio allocation, evaluate diversification, and understand investment risk through an interactive dashboard.

# Overview :

The Portfolio Diversification Analyzer is designed to help users assess how their investments are distributed across different asset classes and evaluate the overall diversification of their portfolio. It provides a structured dashboard interface with secure authentication, allowing users to manage access and interact with portfolio insights in a seamless manner.
The system focuses on delivering a clean user experience while maintaining a scalable architecture for integrating advanced financial analytics such as risk scoring, asset correlation, and portfolio optimization.

# Features :
- User authentication using JWT
- Secure login and signup functionality
- Protected dashboard routes
- Sidebar-based navigation system
- Portfolio and analysis pages
- User information display in settings
- Logout and session handling
- Clean and responsive UI

# Usage :
- Register a new account using the signup page
- Login securely using your credentials
- Access the dashboard to navigate between different sections
- View user details in the settings page
- Logout securely when finished
